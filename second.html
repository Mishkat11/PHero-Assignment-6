Discuss the scope of var, let, and const?

Answer-

Var: Variables declared by var are available throughout the function in which they're declared.

    Let: Variables declared by let are only available inside the block where they're defined.

    Const: It's also like the let variable where Variables declared by const are only available inside the block where they're defined.

Tell us the use cases of null and undefined?

Answer-

`undefined` represents the absence of a value due to variables not being initialized or object properties not being defined.

`null` is used to intentionally indicate the absence of a value or to clear a object property.
